# System Info Tool (Java)

A Java console application that displays important system information.

## Features

- Displays Operating System name and version
- Shows current username
- Shows Java version
- Shows CPU architecture
- Displays total and free RAM

## Technologies Used

- Java
- System properties
- OperatingSystemMXBean
- Console application

## How to Run

Compile:
```
javac SystemInfo.java
```

Run:
```
java SystemInfo
```

## Example Output

```
=== System Information Tool ===
OS: Windows 11
OS Version: 10.0
User: Ava
Java Version: 25
Architecture: amd64
Total RAM: 16384 MB
Free RAM: 8200 MB
```
